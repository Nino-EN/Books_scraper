# 📚 Books Data Analytics Pipeline

A complete end-to-end data project that scrapes book data from a website, cleans and processes it, performs basic statistical analysis, and visualizes insights using a Streamlit dashboard.

---

## 🚀 Project Overview

This project demonstrates a full **data analytics workflow**:

1. **Data Collection** → Web scraping book data from a public website  
2. **Data Cleaning** → Extracting and formatting prices and titles  
3. **Data Analysis** → Calculating key statistics (mean, min, max prices)  
4. **Data Exporting** → Saving results into CSV and Excel files  
5. **Data Visualization** → Interactive dashboard using Streamlit  

---

## 📊 Key Features

- Scrapes 1000+ book records from a live website
- Handles encoding issues and cleans messy data
- Calculates:
  - Average book price
  - Minimum price
  - Maximum price
  - Total number of books
- Exports structured datasets to:
  - CSV format
  - Excel report
- Interactive dashboard with:
  - Price distribution histogram
  - KPI metrics
  - Data preview table

---

## 🛠️ Tech Stack

- Python
- BeautifulSoup (web scraping)
- Requests (HTTP handling)
- Pandas (data analysis)
- Matplotlib (visualization)
- Streamlit (dashboard)
- OpenPyXL (Excel export)
